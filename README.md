# PA9 Minesweeper
Alexander Avecilla, Dominic Oceguera; Lab Section 12

A classic Minesweeper game implementation in C++ using SFML graphics library.

## Game Overview

This project is a simple Minesweeper game built as part of CPT_S 122. Our minesweeper game featyres a graphical user interface using SFML.

## Features

- **Graphical UI**: Rendered with SFML
- **Dynamic Board**: Configurable board sizes with respect to difficulty (Easy, Medium, Hard)
- **Random Mine Placement**: Ensures the first click is always safe unlike the original minesweeper.
- **Game Mechanics**:
  - Left-click to reveal cells
  - Right-click to flag or unflag cells
  - Realt time mine counter and timer
